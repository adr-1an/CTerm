TO SET UP THE PROGRAM, RUN "setup.sh" AS ROOT. (sudo bash setup.sh)
After that, type "./a.out" to run it.
-------------------------------------------------------------------
			    UPDATING CTERM
-------------------------------------------------------------------
Type "system-upgrade" to install the latest CTerm version.
WARNING: That command will reinstall the entire program. It will not check if updates are available first. Every file will be replaced, so if you modified any of them, the changes will be lost.
